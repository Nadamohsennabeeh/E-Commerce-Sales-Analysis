The dashboard is built to foloow up the sales of E-Commerce Platform by using the highly important data fields and dimensions found in the project:

Sales Metrics: Total of Revenue, Orders, and Customers.
Filter bySegmentation: Tracking performance based on and Segments (Home Office, Corporate, Consumer)
Filters by Categories: Breakdown of sales by Category (Electronics, Clothing, Furniture) and Ship Mode (Same Day, first calss, Second Class, Standard Class).
Geographic Coverage: Sales distribution across States and Cities.
Time Intelligence: Analysis of Order Date and Ship Date over months.

Technical Features (Inside the File)
Data Modeling
Custom DAX Measures
Data Cleaning By Power Query
Data visualization by Power BI
